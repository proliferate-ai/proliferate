# Notion research documentation

Use this skill to organize research findings into Notion.

1. Gather source material first, then group it into claims, evidence, implications, and unknowns.
2. Use Notion pages or database rows that match the user's requested location.
3. Keep source links attached to the claims they support.
4. Mark confidence and open questions explicitly.
5. Do not create broad knowledge-base structure unless the user asks for it.
