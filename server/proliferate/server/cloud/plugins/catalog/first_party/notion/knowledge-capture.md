# Notion knowledge capture

Use this skill to turn research, session findings, or decisions into structured Notion content.

1. Identify the target workspace, database, page, or parent location before writing.
2. Extract the durable facts: decision, context, owner, next action, links, and source evidence.
3. Prefer concise headings and bullets over long narrative.
4. When updating Notion, preserve existing page structure unless the user asks for a rewrite.
5. If the target location is unclear, draft the proposed content and ask where it should live.
