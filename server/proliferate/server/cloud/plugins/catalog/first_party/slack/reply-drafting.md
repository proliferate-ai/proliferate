# Slack reply drafting

Use this skill to draft Slack replies from conversation context.

1. Read the target thread or related messages before drafting.
2. Match the requested tone and audience. Keep the draft short unless the user asks for a longer response.
3. Include unresolved assumptions below the draft when they matter.
4. Do not send the draft. Return it to the user unless they explicitly ask you to send and the mounted tools support sending.
5. If the requested reply needs private or sensitive information, call that out instead of inventing details.
