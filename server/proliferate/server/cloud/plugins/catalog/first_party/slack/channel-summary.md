# Slack channel summary

Use this skill to summarize Slack channel or thread activity.

1. Resolve the channel, thread, topic, or timeframe.
2. Fetch the smallest useful slice of messages and thread replies.
3. Summarize decisions, open questions, owners, deadlines, and links separately.
4. Include source references using channel names, timestamps, or permalinks when available.
5. Do not infer agreement from silence. Mark uncertain or conflicting messages clearly.
