# Gmail read-only lookup

Use this skill for read-only Gmail search and summarization.

1. Search narrowly by sender, recipient, subject, date range, labels, or terms.
2. Fetch only the messages or threads needed for the task.
3. Summarize senders, dates, requested actions, attachments, and unresolved questions.
4. Do not send, archive, delete, label, mark read, or otherwise modify mail. Proliferate launches Gmail with read-only scope.
5. Avoid exposing private email content beyond what is needed to answer the user.
