# Gmail inbox triage

Use this skill to triage Gmail threads without changing mailbox state.

1. Define the triage window or label set before searching.
2. Group messages by urgency, sender, thread, and required response.
3. Mark which items are informational, blocked, waiting on the user, or likely actionable.
4. Draft suggested replies only when useful, but do not send them.
5. Preserve message or thread ids returned by tools for follow-up.
