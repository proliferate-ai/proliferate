# Gateway (`@proliferate/gateway`)

WebSocket gateway that handles all real-time streaming between clients and sandboxes (Client ↔ Gateway ↔ Sandbox).

- Spec: `SPEC.md`
- Docs: https://docs.proliferate.com/introduction/core-concepts

## Development

- From repo root: `pnpm dev:gateway`
- Tests: `pnpm --filter @proliferate/gateway test`
