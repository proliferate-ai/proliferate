# Product Agent Features

This platform area owns the reusable Product MCP server pattern and the target
definitions exposed through it.

- [Product MCP Servers](servers.md) — server ownership, registration, tool
  boundaries, and runtime integration.
- [Product MCP Definitions](definitions/README.md) — individual target
  definitions for product-owned agent capabilities.

User-facing workflows that consume these capabilities remain under
[Product Systems](../../../systems/product/README.md).
