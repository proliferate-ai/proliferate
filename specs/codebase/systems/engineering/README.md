# Engineering Systems

Engineering systems are complete delivery and operational domains.

- [Analytics](analytics/README.md) — product/adoption measurement, anonymous
  telemetry, PostHog routing, and durable Metabase facts and views.
- [Delivery](delivery/README.md) — release and update behavior.
- [Issue Lifecycle](issue-lifecycle/README.md) — support ingestion,
  investigation, resolution, and communication.
- [Observability](observability/README.md) — event production, scrubbing,
  correlation, structured logs, Sentry, and infrastructure signals.

Contributor procedures for operating these systems live under
[Developing](../../../developing/README.md).
