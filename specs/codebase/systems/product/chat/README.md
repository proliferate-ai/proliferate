# Chat Systems

Start with the product task you intend to change:

| Task | Owner |
| --- | --- |
| Create, update, or replace a visible chat | [Chat lifecycle](lifecycle.md) |
| Change input, controls, picker presentation, panels, or badges | [Composer](composer.md) |
| Change messages, streaming, replay, rows, or long-history behavior | [Transcript](transcript.md) |
| Change tab ordering, restoration, or projected-shell mechanics | [Workspaces](../workspaces/README.md) |
| Change model identity, availability, or selection action classification | [Model Catalog](../../../platforms/product/model-catalog.md) |
| Change session actors, live config, process retirement, or relaunch | [AnyHarness sessions](../../../structures/anyharness/src/sessions.md) |
| Decide where frontend source belongs | [Frontend structure](../../../structures/frontend/README.md) |

Each focused document owns its listed product behavior and links to its
dependencies. Do not copy runtime, workspace-shell, catalog, or source-layout
contracts into the Chat documents.
